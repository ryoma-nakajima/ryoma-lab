RYOMA LAB

実験的 Web ツール。

単一 HTML で完結する小さな道具を、検証しながら少しずつ並べていく場所。気に入ったものはそのまま使えます。すべてのソースは GitHub で公開しています。

GitHub でソースを見る ryoma-nakajima/ryoma-lab
TOOL-01

テロップアニメーション プリセット集

動画編集向けのテキストアニメーション 60+ 種。enter → hold → exit の 3 秒ループでプレビューできるリファレンス。

NOTE-01

Claude Code × Codex 並列実装ワークフロー

実装は Codex に並列ファンアウト、Opus が司令塔。Claude Code の Workflow 設定一式（codex-impl.js + CLAUDE.md の仕込み方）を丸ごと無料公開。

読む→
TOOL-02

映像エフェクト大全

Premiere / After Effects / DaVinci / CapCut ほか、世の中の動画編集ツールに実在する映像エフェクトを 5,500+ 種・5 系統で網羅した検索リファレンス。

TOOL-03

エフェクト・プレイグラウンド

映像エフェクト 288 種を WebGL でリアルタイム実装。スライダーで挙動を調整しながら、各エフェクトが実際にどう動くかを確認。自分の動画/画像をドロップして適用も可能。

TOOL-04

トランジション・プレイグラウンド

動画トランジション 695 種を WebGL でリアルタイム実装。30 ファミリーに分類し、スライダーで A→B 遷移の挙動を調整。タイル一覧から実際の動きを一望できる。
